# Prompts y dirección visual

## Prompt base para landing / mockup

Crear una landing page para la conferencia “Por el Poder del Espíritu Santo”, manteniendo la identidad visual del carrusel: fondo oscuro navy/charcoal, tipografía sans-serif moderna, paleta blanco + naranja/dorado, llama dorada del lado derecho, composición sobria, minimalista y premium.

La landing debe incluir:
- Header con logo y navegación.
- Hero con título, fecha, ubicación y CTA.
- Sección “Por qué esta conferencia”.
- Sección de oradores con foto, nombre y minibio.
- Cronograma dividido por viernes 11 y sábado 12.
- Información práctica.
- Bloque de registro con CTA.
- Preguntas frecuentes.
- Footer.

No usar fondos blancos. No usar estilos ajenos a las piezas gráficas. No sobrecargar con elementos decorativos. Priorizar legibilidad, sobriedad y claridad.

## Prompt para mockup visual

Diseñar un mockup visual de landing page desktop para una conferencia cristiana titulada “Por el Poder del Espíritu Santo”. Usar fondo oscuro navy/charcoal, títulos naranja/dorado, texto blanco, cards oscuras con borde dorado sutil y una llama dorada abstracta como elemento visual. Mantener estética sobria, moderna, ministerial y premium. Incluir secciones: hero, propósito, oradores, cronograma, información práctica, registro y footer.

## Prompt para generar assets de fondo

Crear un fondo oscuro navy/charcoal con textura sutil y una llama abstracta dorada/naranja en el lado derecho. Estilo gráfico, sobrio, minimalista, premium. Sin texto, sin logos, sin personas. Debe funcionar como fondo de una landing page.
