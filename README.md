# Paquete de diseño — Por el Poder del Espíritu Santo

Este ZIP contiene el design system, contenido, cronograma, oradores, prompts y assets disponibles para trabajar la landing de la conferencia.

## Estructura

- `01_design-system/`
  - Guía visual y componentes web.
- `02_content/`
  - Contenido base de la landing.
  - Oradores.
  - Cronograma.
  - Datos estructurados en JSON.
- `03_prompts/`
  - Prompts para mockups y generación visual.
- `04_assets/`
  - Logo, fotos de oradores, imágenes de referencia y mockup generado.
- `05_source-documents/`
  - Documento original del cronograma.
